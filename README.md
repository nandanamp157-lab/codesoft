# Titanic Survival Prediction

## Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning. It uses the Titanic dataset and a Random Forest Classifier to classify passengers as survived or not survived.

## Dataset
The dataset contains passenger information such as:
- Passenger Class (Pclass)
- Sex
- Age
- Fare
- Embarked
- SibSp
- Parch
- Survival Status (Target)

Dataset Source:
https://www.kaggle.com/competitions/titanic/data

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Project Workflow
1. Load the dataset
2. Handle missing values
3. Encode categorical features
4. Split the data into training and testing sets
5. Train the Random Forest model
6. Evaluate the model using Accuracy, Classification Report, and Confusion Matrix

## Results
- Accuracy: Approximately 82% 
- Classification Report
- Confusion Matrix
- Feature Importance
